─────────────────────────────────────────────────────────────
  HOW TO DROP FILES
─────────────────────────────────────────────────────────────

   1. Just drag-and-drop your images into this folder:
      /source-images/

   2. Naming is optional — you can name them anything.
      If you do name them with the filenames above, I can
      place them automatically.

   3. Once dropped, tell me "images are in" and I will:
      • pick the best shot for each slot
      • crop to the correct aspect ratio
      • optimise file size (keeping quality)
      • move to /assets/images/ with the right filename
      • update the HTML if any image is better-suited to a
        different slot than I first planned
